# Chatbot HTML – OpenRouter (TP)

## Objectif
Réaliser un chatbot web simple dans le cadre d’un TP, en lien avec la plateforme OpenRouter.

## Technologies
- HTML
- CSS
- JavaScript
- GitHub Pages

## Remarque
L’API OpenRouter nécessite une clé privée et un système de paiement.
Dans ce TP, le chatbot est implémenté côté front-end avec des réponses simulées
afin de démontrer le fonctionnement logique d’un chatbot.

## OpenRouter
OpenRouter est une plateforme qui permet de connecter des chatbots à différents modèles d’intelligence artificielle via une seule API. 
Dans ce TP, le chatbot fonctionne côté front-end avec des réponses simulées. 
OpenRouter est mentionné pour expliquer le fonctionnement d’un vrai chatbot IA.

## Schéma du fonctionnement du chatbot

Utilisateur → Chatbot HTML → Simulation / Théorie OpenRouter → Réponse
